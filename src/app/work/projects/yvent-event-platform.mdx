## Overview

YVENT is a centralized platform designed to streamline event access and management for students. Built with a team of four developers, the platform significantly improved event participation rates by providing a unified interface for discovering and registering for campus events.

## Key Features

- **Centralized Event Management**: Created a single platform for all student events, making it easier for students to discover and participate in activities.
- **Responsive Design System**: Implemented a comprehensive design system ensuring consistent user experience across all devices.
- **Authentication System**: Developed a secure authentication system for user management and event registration.
- **Real-time Updates**: Integrated real-time notifications for event changes and updates.

## Technologies Used

- **React & Next.js**: For frontend development and server-side rendering
- **Tailwind CSS**: For responsive and modern UI design
- **Authentication System**: Custom authentication system
- **State Management**: Implemented efficient state management for real-time updates
- **Strapi**: For content management and API integration.
- **Scalingo**: For deployment and hosting.

## Challenges and Learnings

The main challenge was creating a scalable architecture that could handle multiple events and user interactions simultaneously. This was addressed through careful planning of the data structure and implementing efficient state management. Another challenge was ensuring the platform was accessible to all students, which was solved through responsive design and user testing.

## Outcome

YVENT successfully increased student participation in campus events by 40%, demonstrating the platform's effectiveness in connecting students with relevant activities. The project showcased the importance of user-centered design and efficient technical implementation in creating impactful solutions for the student community.
